Документация по Генератору Паролей

Файл: PasswordGenerator.h

Заголовочный файл генератора паролей.

Содержит определение класса PasswordGenerator.

Класс: PasswordGenerator

Класс для генерации случайных паролей.

Поддерживает три уровня сложности: простой, средний и сложный.

Публичные типы

Перечисление: ComplexityLevel

Уровни сложности пароля.

Simple = 1

Простой уровень (только буквы)

Medium

Средний уровень (буквы и цифры)

Strong

Сложный уровень (буквы, цифры и символы)

Публичные методы

std::string generatePassword(int level)

Генерирует пароль заданной сложности.

Параметры:

  • level – уровень сложности (1-3)

Возвращает: Сгенерированный пароль

Исключения:

  • std::invalid_argument — если уровень некорректен

Приватные методы

bool isValidComplexityLevel(int level) const

Проверяет допустимость уровня сложности.

std::string getCharsetForLevel(int level) const

Возвращает набор символов для заданного уровня сложности.

size_t getPasswordLengthForLevel(int level) const

Возвращает длину пароля для заданного уровня сложности.

char generateRandomChar(const std::string &charset) const

Генерирует случайный символ из заданного набора.
